## Deploy in 60 Seconds

**Vercel (recommended)**
1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com) → New Project → Import repo
3. Click Deploy — done

**Netlify**
1. Go to [netlify.com](https://netlify.com) → Add new site
2. Drag and drop the `index.html` file
3. Live instantly

**GitHub Pages**
1. Go to repo Settings → Pages
2. Set source to `main` branch → `/ (root)`
3. Your site will be live at `https://Adamya246.github.io/portfolio`

## Customization

All content is in `index.html`. To update:

- **Name / tagline** — edit the hero section
- **Projects** — find the `#projects` section, update title, description, and stack tags
- **Links** — search for `href=` to find and update all social/project links
- **Photo** — replace the base64 image src with a new one, or swap to a file path after deploying
- **Colors** — edit the CSS variables at the top of the `<style>` block:

```css
:root {
  --accent1: #ff4d00;  /* orange */
  --accent2: #00e5ff;  /* cyan   */
  --accent3: #b8ff00;  /* lime   */
  --accent4: #ff00aa;  /* pink   */
}
```
